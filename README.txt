WorkShopErp project for IMIE (Rennes)

Autor : Benjamin Thiebaut (DL)

Before start : check if your database is ON and configurated : 

	- Open the file "WorkShopErp/WorkShop/Enums/DataConnectionRessouce.cs"
	- Replace the connection string "Server=127.0.0.1;Port=3306;Database=csharp_workshop;Uid=root;Pwd=;"" with youre own connection string.
	- Check the startup project to start WebApplication1 without debug then WorkShopWpf (with debug as you want)


Executable can be found in "WorkShopCSharp\WorkShopWpf\bin\Release" location.

Required configuration : 
	- Windows 10
	- 500 MB Free Ram
